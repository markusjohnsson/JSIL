This demo is a simple XNA 4.0 game, compiled from C# to JavaScript using JSIL, my open-source C# to JavaScript compiler. It should run in any browser that supports HTML5 and a modern version of JavaScript, no plugins required.

You can get the source code to this demo from the author's page on GitHub.

To begin playing, click the Load Game button below.

Left/right arrows to move piece. Up to rotate piece. Space to drop piece.
